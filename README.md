# aca-mcp-server
A demo MCP server that handles team members and their skills

## Development Environment

This project includes a devcontainer configuration for a consistent development experience. The devcontainer includes:

- **Azure Developer CLI (azd)** - For Azure deployment and management
- **Azure CLI** - For Azure resource management  
- **Docker** - For containerization
- **Node.js 20** - JavaScript runtime
- **Python 3.11** - Alternative runtime
- **Git & GitHub CLI** - Version control tools

### Getting Started

1. Open this repository in VS Code
2. When prompted, click "Reopen in Container"
3. Wait for the container to build (first time may take a few minutes)
4. Run `.devcontainer/verify-tools.sh` to verify all tools are installed
5. Start developing!

See [.devcontainer/README.md](.devcontainer/README.md) for detailed information about the development environment.
